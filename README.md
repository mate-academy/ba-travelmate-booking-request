# Travel Mate Web Site Landing Page

Welcome to the Travel Mate landing page project!

**Description:**
TravelMate is a learning project for the Mate academy business analysis course.

## How to run the website locally

Here are step-by-step instructions to set up the necessary libraries and packages and start the project locally, on both newly set-up Mac and Windows laptops.

### Instructions for Mac OS

1. **Install Homebrew** (if not already installed). Open your Terminal and run:
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
2. **Install Node.js** using Homebrew by typing:
   ```bash
   brew install node
   ```
3. **Navigate to the project folder:**
   ```bash
   cd "path/to/mate-academy-travel-mate"
   ```
4. **Install all needed libraries and packages:**
   ```bash
   npm install
   ```
5. **Start the local development server:**
   ```bash
   npm run dev
   ```
6. **View the website:** Open your browser and go to the link provided in your terminal (usually `http://localhost:5173`).

### Instructions for Windows OS

1. **Install Node.js:** Download and install the LTS version of Node.js from [nodejs.org](https://nodejs.org/). This installer automatically includes npm (node package manager).
2. **Open Command Prompt or PowerShell** and **navigate to the project folder:**
   ```cmd
   cd "path\to\mate-academy-travel-mate"
   ```
3. **Install all needed libraries and packages:**
   ```cmd
   npm install
   ```
4. **Start the local development server:**
   ```cmd
   npm run dev
   ```
5. **View the website:** Open your browser and go to the link provided in your terminal (usually `http://localhost:5173`).
